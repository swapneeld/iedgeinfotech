Web Development Course Schedule

15 Days Intensive Hands-on Training

Week 1: Introduction, MVC Basics, and Models

  • Day 1: Introduction to Web Development & .NET MVC (1 hour)
    • Overview of Web Development
    • Introduction to the MVC Pattern
    • Setting up the Development Environment (Visual Studio 2022)
  • Day 2: Project Structure and MVC Architecture (1 hour)
    • Understanding Models, Views, Controllers
    • Creating Your First MVC Application
  • Day 3: Routing and Controllers in .NET MVC (1 hour)
    • Detailed Explanation of Routing
    • Creating and Managing Controllers
  • Day 4: Introduction to Models and Data Access (1 hour)
    • Creating Models
    • Connecting to a Database using Entity Framework Core 7.0
  • Day 5: Data Validation and LINQ (1 hour)
    • Implementing Data Annotations for Validation
    • Introduction to LINQ for Data Manipulation

Week 2: Views, jQuery Integration, and APIs

  • Day 6: Views and Razor Syntax (1 hour)
    • Creating Views in .NET MVC
    • Using Razor Syntax
  • Day 7: Form Handling and HTML Helpers (1 hour)
    • Working with HTML Helpers
    • Handling Forms in MVC Applications
  • Day 8: Client-Side Scripting with jQuery (1 hour)
    • Integrating jQuery 3.6
    • Basic jQuery Syntax and Selectors
  • Day 9: AJAX, Partial Views, and Hands-On Lab (1 hour)
    • Implementing AJAX Calls
    • Creating and Using Partial Views
    • Lab: Building Forms with jQuery and AJAX
  • Day 10: Introduction to Web APIs in .NET (1 hour)
    • Creating Web APIs
    • Consuming APIs in MVC Applications

Week 3: Advanced Topics, Security, and Final Assessment

  • Day 11: Dependency Injection (DI) and Unit Testing (1 hour)
    • Overview of Dependency Injection and its importance in MVC
    • Writing Unit Tests for Controllers and Services
  • Day 12: Securing Web APIs and Authentication (1 hour)
    • Implementing Authentication and Authorization using ASP.NET Identity
    • Best Practices for API Security
  • Day 13: Advanced Concepts and Hands-On Lab (1 hour)
    • Students create and secure APIs within their MVC application
  • Day 14: Advanced Error Handling, Logging, and Project Development (1 hour)
    • Implementing advanced error handling
    • Integrating a logging framework like Serilog
    • Students work on finalizing their projects
  • Day 15: Project Presentation and Assessment (1 hour)
    • Students present their final MVC application project
    • Course Assessment and Feedback

Tools and Versions

  • .NET MVC: Version 7.0
  • Entity Framework: Core 7.0
  • Visual Studio: 2022
  • jQuery: Version 3.6
  • C#: Version 10
